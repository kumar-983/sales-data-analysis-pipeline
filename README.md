# Sales Analysis Project

## Overview
End-to-end sales analytics project built using Python, Excel, SQL, and Power BI.

This project demonstrates data cleaning, KPI analysis, lookup operations, pivot analysis, and dashboard visualization.

---

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Excel
- SQL
- Power BI

---

## Project Workflow
Raw Data → Data Cleaning → KPI Analysis → Pivot Analysis → Dashboard Visualization

---

## Key Business Insights
- Technology generated highest revenue
- High discount levels reduced profitability
- Top customers contributed major revenue share
- Regional performance varied significantly

---

## Project Components

### Python
Data cleaning and preprocessing

### Excel
- KPI calculations
- XLOOKUP / VLOOKUP
- Pivot analysis

### Power BI
Interactive executive dashboard
